Tic Tac Toe game (C Language)

This project is a simple implementation of the Tic Tac Toe game in C, allowing two players to compete against each other.
It provides features like player name input, symbol assignment, interactive grid, win and draw detection,
game statistics, and replay option. Players can enjoy the classic game of Tic Tac Toe on the console.

  • Features :

    1. Two-player gameplay: The game allows two human players to take turns and compete against each other.
    2. Player names: Players can enter their names at the beginning of the game, which are stored and used throughout the gameplay.
    3. Symbol assignment: Player 1 is assigned the symbol 'X', and Player 2 is assigned the symbol 'O'.
    4. Interactive grid: The players mark their moves on the grid by entering corresponding alphabets from the keyboard.
    5. Validity checks: The program checks for invalid inputs, such as marking a cell that is already occupied.
    6. Winning condition: The game determines the winner by checking for three identical symbols in a row, column, or diagonal.
    7. Draw detection: If all cells are filled and no winner is determined, the game declares a draw.
    8. Game statistics: The program keeps track of the number of games played, wins for each player, and draws.
    9. Rules and instructions: Players have the option to read the rules and instructions of the game before starting.
    10. Replay option: After completing a game, players can choose to continue playing with different players or stop the game.
